## Install
- Clone the repo:
    ``` sh
    git clone --recursive https://github.com/xuan3986/UDDETTS.git
    cd UDDETTS
    git submodule update --init --recursive
    ```

- Create Conda env:

    ``` sh
    conda create -n UDDETTS -y python=3.8.20
    conda activate UDDETTS
    pip install -r requirements.txt
    ```
- Model download:
